## How to play

- download the zip, unpack and run "Myyra.exe"

### Controls

- LEFT/RIGHT ARROW : switch side on the root and control in mid-air
- UP/DOWN ARROW    : slow down or speed up the sliding
- SPACE            : jump of the root
- Esc              : Pause
